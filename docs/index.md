# Solo mining:
create in data folder - file apix.conf
```
rpcallow=127.0.0.1
server=1
daemon=1
rpcuser=user
rpcpassword=x
port=9661
rpcport=9660
```

In bat file cpuminer-opt-apx
```
cpuminer-aes-sse42.exe -a yespowerr16 -o http://127.0.0.1:9660 -u user -p x --coinbase-addr=WALLET_ADDRESS
```
